Every time you have a new COA to add:

Step 1 — Upload the file to Google Drive

Open Google Drive → VPN COA Database → click into the vendor folder (e.g. LiliPeptide Luna)
Drag the COA image/PDF in
Step 2 — Make the folder shareable (one time per folder)

Right-click the vendor folder → Share → set to "Anyone with the link can view"
You only do this once per folder — new files inside automatically inherit it
Step 3 — Go to the website

Click ⬆ Upload COA — Admin on the main page
Enter the admin password (ask David or Said Kol)
Step 4 — Scan the folder

Select the vendor from the dropdown (e.g. LiliPeptide Luna)
Right-click the vendor folder in Drive → Copy link → paste it into the Folder Auto-Import box
Click 🔍 Scan Folder
Step 5 — Confirm and submit

The site shows every file it found with the peptide auto-detected
Fix any wrong ones using the dropdowns
Click Submit All
That's it. Next time you add more COAs to the same folder, just repeat Steps 3–5 — the scan will pick up any new files automatically.
