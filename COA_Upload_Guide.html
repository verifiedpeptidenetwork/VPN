COA Upload Guide

How to add lab test results to the database · Admin use only

Overview

This guide explains how to upload a Certificate of Analysis (COA), endotoxin test, or heavy metals test to the VPN database. The process has three parts: saving the file to Google Drive, copying its link, and pasting it into the website's upload panel.

Who can do this: Only admins (David or Said Kol). Members can view COAs but cannot upload them.

Before You Start

✓
What You Need
  • Access to Google Drive (VPN COA Database folder is already set up)
  • The COA file — a photo or PDF of the lab test document
  • The admin password — ask David or Said Kol
  • The VPN website open in your browser

Step-by-Step Instructions

1
Upload the File to Google Drive
  • Open Google Drive and go to the VPN COA Database folder
  • Click into the correct vendor subfolder (e.g. Kerui Peptides, Sophie, Anya, etc.)
  • Drag your COA file directly into that folder — or click New → File Upload
  • Wait for the upload to finish (you'll see a checkmark)
Tip: Keep the original filename — you can rename it later if needed. The name does not affect how it appears on the site.
2
Get the Shareable Link
  • Right-click the file you just uploaded in Drive
  • Click Share → Copy link
  • Make sure it says "Anyone with the link can view"
  • If it says "Restricted" — click the dropdown and change it to Anyone with the link
  • Click Copy link — the link is now on your clipboard
Important: If this is set to "Restricted", members will get an Access Denied error when they try to view the COA. Always set it to "Anyone with the link."
3
Open the Admin Upload Panel on the Website
  • Go to the VPN website main page
  • Click the ⬆ Upload COA — Admin button (above Meet The Researchers)
  • A password prompt will appear — enter the admin password
  • The Batch COA Upload panel will open
4
Fill in the Upload Form

Once the panel is open:

  • Select the Vendor from the dropdown — match it to the Drive folder you used
  • Enter the Lab Name (e.g. Janoshik) — optional but recommended
  • Enter the Batch / Lot # if visible on the document — optional
  • Choose the tab: Vendor Testing or Independent Testing
  • Find the peptide name in the list (scroll to find it)
  • Click the correct column (COA, Endotoxin, or Heavy Metals) and paste the Drive link
  • Repeat for any other peptides you have files for
  • Click Submit All
Tip: You can paste multiple links in one session — one per peptide row. You don't have to submit after each one. Fill everything in first, then hit Submit All.
5
Done — It's Live Immediately

The COA is now on the website. Members will see a 👁 View button next to that peptide. When they click it, the document opens directly from Google Drive — no login required on their end.

Vendor Testing vs Independent Testing

The upload panel has two tabs. Use the correct one depending on who ordered the test:

Tab When to use it Example
Vendor Testing The vendor (Kerui, Sophie, etc.) provided the COA document themselves Kerui sends you their Janoshik COA for Retatrutide
Independent Testing The VPN group independently ordered and paid for the lab test The inner circle sends a peptide to Janoshik themselves and receives the results

Test Type Reference

Column What it tests Lab method
COA Purity of the peptide — confirms it is what it says it is and how pure it is HPLC, Mass Spec
Endotoxin Tests for bacterial toxins (endotoxins) that can cause serious reactions LAL or rFC test
Heavy Metals Tests for dangerous metals like lead, arsenic, mercury, and cadmium ICP-MS analysis

Rules to Always Follow

Rule Why It Matters
Always set Drive files to "Anyone with the link can view" Members get an Access Denied error if this is set to Restricted
Upload to the correct vendor subfolder in Drive Keeps the Drive organized and easy to find files later
Match the vendor folder to the vendor you select in the upload form Ensures the COA appears under the right vendor on the website
Only use Google Drive links — not Dropbox, OneDrive, or direct file uploads The site is built to embed Google Drive previews specifically
Never share the admin password with members Only admins should be able to add or delete COAs

What Members See

Members never interact with Google Drive or the upload form. Here is their experience:

→

Member opens the website → clicks a vendor → scrolls to a peptide → sees 👁 View next to a test type → clicks it → the COA document opens in a full-screen viewer, embedded directly from Google Drive. No Google account needed, no login, no downloading.
