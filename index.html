Database Setup
Step 1 — Create Free Supabase Account
1. Go to supabase.com → Sign up free
2. Create a new project (name it anything)
3. Wait ~2 minutes for it to provision
Step 2 — Create the Reports Table
In Supabase → SQL Editor → paste and run this:
create table reports ( id uuid default gen_random_uuid() primary key, created_at timestamptz default now(), vendor_id text, peptide text, test_type text, file_name text, file_data text, lab text, batch text, notes text ); alter table reports enable row level security; create policy "Public read" on reports for select using (true); create policy "Public insert" on reports for insert with check (true);
Step 3 — Get Your Keys
In Supabase → Project Settings → API:
Copy your Project URL
Copy your anon public key
Step 4 — Paste Keys into the HTML File
Open your index.html in a text editor
Find these two lines near the top of the script:

const SUPA_URL = '';
